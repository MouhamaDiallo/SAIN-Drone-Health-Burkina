# SAIN-Drone-Health-Burkina
Analyse de la répartition des infrastructures de santé au Burkina Faso et proposition d’une solution logistique par drones.


Objectif  
Ce projet vise à analyser la répartition des infrastructures de santé afin d’identifier les zones sous-desservies et proposer une solution de livraison de médicaments par drones.

Données  
Les données proviennent de la plateforme Humanitarian Data Exchange (HDX) :  
https://data.humdata.org/dataset/burkina-faso-health-facilities  

Le dataset comprend 951 structures de santé géolocalisées sur l’ensemble du territoire.

Méthodologie  
Les données géospatiales (shapefile) ont été traitées avec R afin d’extraire les coordonnées et nettoyer les variables. Elles ont ensuite été transformées en format CSV, compatible avec les outils de visualisation.  
L’analyse et les visualisations ont été réalisées avec Power BI.

Résultats  
L’analyse met en évidence une forte concentration des infrastructures dans les zones centrales et nord du pays, tandis que les régions de l’est et du sud présentent une densité plus faible.  
Les hôpitaux représentent la majorité des structures recensées.

Proposition  
Sur la base de ces résultats, une solution de livraison de médicaments par drones est proposée afin d’améliorer l’accès aux soins dans les zones les moins desservies.
